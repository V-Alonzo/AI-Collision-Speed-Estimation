Actúa como un experto en diseño de prompts y procesamiento de información con modelos de lenguaje como ChatGPT, 
considerando sus fortalezas, limitaciones y mejores prácticas.

Se está desarrollando un archivo en formato JSON cuyo propósito es definir de manera precisa qué información debe 
extraerse de reportes de accidentes de tránsito en formato PDF. Esta información será utilizada posteriormente para 
entrenar un modelo de machine learning que permita estimar la velocidad del vehículo involucrado antes del impacto.

Tu tarea es analizar el JSON proporcionado, incluyendo su estructura, organización y los campos definidos.

Con base en este análisis:

Propón una reestructuración del JSON que mejore la claridad, consistencia y facilidad de uso para un modelo de lenguaje 
encargado de extraer la información desde documentos PDF.

Optimiza la organización de los campos para que sean fácilmente interpretables y reduzcan ambigüedades durante la extracción.
Asegúrate de que la estructura resultante facilite la identificación contextual de cada dato dentro del reporte.

No agregues información nueva ni inventes campos que no estén presentes en el JSON original; únicamente reorganiza, renombra
 (si es necesario) y mejora la estructura existente.

El objetivo final es que el JSON funcione como una guía clara, precisa y eficiente para orientar a un modelo como ChatGPT en 
la extracción automatizada de datos relevantes desde los reportes.
